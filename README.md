# deepfake-detector
# Deepfake Detection using Deep Learning

## Overview
This project detects whether an image or video is real or manipulated using deep learning techniques.

## Features
- Upload image/video for analysis
- Detect deepfake content
- Display prediction confidence score
- User-friendly web interface

## Tech Stack
- Python
- TensorFlow/PyTorch
- OpenCV
- Flask/Streamlit
- HTML, CSS, JavaScript

## Dataset
FaceForensics++, Celeb-DF, DFDC
